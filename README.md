# Pasidi Sarangi - Portfolio

Welcome to my personal portfolio website! I am a final-year BSc Hons in Information Technology student at SLIIT, bridging user experience design and software development.

## 🚀 Features
- **Responsive Design**: Fully responsive layout that looks great on desktop, tablet, and mobile devices.
- **Modern UI/UX**: Built with a sleek, dark-themed glassmorphism aesthetic, featuring micro-interactions and smooth animations.
- **Dynamic Projects**: Showcases my latest work in Software Engineering, UI/UX Design, and Business Analysis.
- **Interactive Elements**: Typing text effects, floating 3D elements, and animated scroll indicators.

## 🛠️ Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Custom Variables, Flexbox/Grid, Animations, Glassmorphism)
- JavaScript (Vanilla JS for DOM manipulation, Intersection Observers for scroll animations)
- Font Awesome (Icons)
- Vite (Development Server & Bundler)

## 💻 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/PasidiSarangi/My-Portfolio.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd My-Portfolio
   ```

3. **Install dependencies (if applicable):**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open in Browser:**
   Open your browser and navigate to the local host address provided by Vite (usually `http://localhost:5173`).
